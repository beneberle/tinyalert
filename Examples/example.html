TinyAlert

Tiny and simple alert with Mootools and CSS3 only. (js only ~1.2k)


Examples:

Basics:

  • Simple Alert
  • Top Left
  • Bottom Left
  • Bottom Right
  • Stack
  • Custom duration

Skins:

  • Blue Skin
  • Snow Skin
  • Gray Skin
  • Black Skin

Icons:

  • Alert with icon

Callbacks:

  • Alert with callback (click on alert)

How to use:

var b = new TinyAlert(options);

Options:
el - Element where Bubble will be inject. Default document.body
position - Position to show tr,tl,br,bl. Default tr (Top Right)
delay - Duration time. Default 3000ms
skin - Bubble skin. Default clean

Example:
var b = new TinyAlert({delay:3000, skin:'black'});

Methods:
show(title, message, icon, callback)

Simple example:
var b = new TinyAlert();
b.show('Title','Your message');

Dependences:

  • Mootools 1.2.4 Core
  • tiny_alert.js (~1.2k)
  • tiny_alert.css (~3.2kb with 4 skins)

Creating your skins

Styles:

/*Main TinyAlert block style.*/
#skbubble > div.skinName {
}

/*Title style.*/
#skbubble > div.skinName > span.tatitle{
}

/*Message style.*/
#skbubble > div.skinName > span.tamsg{
}
